# html5_features_detect
A list of functions used to detect HTML5 featues
## Usage
```javascript
var support = require('support');
//support history pushState
if(support.history) {
  //your code 
}
```
### html5 features detect
1. location    
2. input        
3. placeholder  
4. autofocus    
5. history      
6. canvas       
7. video        
8. localStorage 
9. webworkers   
10. offline
